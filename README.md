# freecad-panel-feature
FreeCad addon to create wood panels for cabinet construction

Creates a parametric panel FeaturePython object:
- Size: Length, Width, Thickness
- BaseColor for top face
- Optional bottom coloring/texture
- ABS edge banding visualization via per-face color/texture (view-only)

## Install
- Tools → Addon Manager → (after submission) search "PanelFeatureAddon"
  or
- Addon Manager → (if using custom repo) add repository URL
  or
- Manual: copy this folder into FreeCAD Mod folder.
